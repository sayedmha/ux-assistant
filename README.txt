# UX Assistant - مساعد تجربة المستخدم

هذا السكربت يظهر مساعدًا ذكيًا للمستخدم إذا لم يتفاعل مع الصفحة لفترة معينة.

## الاستخدام

1. أضف السطر التالي داخل موقعك:
<script>
  window.UX_OPTIONS = {
    language: 'ar',
    themeColor: '#e63946',
    idleTime: 15000
  };
</script>
<script src="https://YOUR-LINK/ux-assistant.js"></script>

2. غير القيم حسب حاجتك.

## الميزات

✅ يظهر زر مساعدة ثابت  
✅ يظهر مساعد عند الخمول  
✅ لا يحتاج خادم أو قواعد بيانات
